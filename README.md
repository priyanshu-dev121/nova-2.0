# Campus Nova

A comprehensive MERN-stack platform for campus attendance, resource management, and incident resolution.

## Project Structure

- **frontend/**: React + Vite application (Student & Faculty Dashboards).
- **backend/**: Node.js + Express API (MongoDB, JWT Auth, Cloudinary).

## How to Run

1. **Backend**:
   ```bash
   cd backend
   npm run dev
   ```

2. **Frontend**:
   ```bash
   cd frontend
   npm run dev
   ```
